# ConnectUtahToday

## Project Structure

- `apps/website` — Website frontend and PayloadCMS backend
	- `apps/website/app/(app)` — NextJS pages
	  ```
	  const PageName = () => {}

	  export default PageName;
	  ```
	- `apps/website/app/(payload)` — PayloadCMS pages autogenerated from the PayloadCMS configuration, not to be edited
	  directly.
	- `apps/website/components` — React components
	  ```
	  export const ComponentName = () => {}
	  ```
